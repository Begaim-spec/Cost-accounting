# Web-приложения для учета расходов


## Функционал сайта
  - Ввести имя пользователя;
  - Добавление названия расхода и сумму;
  - Удаление расхода;
  - Сбросить всё;
  - Сортировка по возрастанию и убыванию



### Запуск проекта 

Для запуска проекта вам необходимо  набрать команду npm/yarn start 

### Технлогический стек: 
  - REACT;
  - REDUX;
  - JAVASCRIPT;
  - MATERIAL UI.
